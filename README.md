# TOTP-QR-Code-Compliance-Test
I needed a simple set of tests to check compliance of a QR Code to troubleshoot a 2FA Plugin.  Note that this work not been independently reviewed or checked for accuracy or completeness.

Pre-requisites:
sudo apt-get install zbar-tools 
Instructions:
Run this script and the script will enable you to select any image contained in the current folder as input.  
Results will be listed to screen.
